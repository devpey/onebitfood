# Onebitfood

## _MVP baseado no IFood._
## _Projeto desenvolvido na 5ª Semana FullStack do onebitcode onebitcode.com._

## Ferramentas utilizadas

- Ruby 3.0.0
- rails 6.0.4.1
- SQLite3

## Instalação

Instale as gems
```sh
bundle install
```
Crie o bancos de dados e suas tabelas

```sh
rails db:create
rails db:migrate
```
Popule o banco de dados 
```sh
rails db:seed
```

Inicie o servidor
```sh
rails s
```
